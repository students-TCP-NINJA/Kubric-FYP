# Visio Import Notes

## Inputs in this repo

- Draw.io files in `diagrams/drawio`
- Mermaid and PlantUML sources for regeneration

## Recommended workflow

1. Convert Draw.io to SVG using diagrams.net export.
2. Import SVG to Visio and align to a custom page grid.
3. Apply standard Visio network stencils where symbols are required.
4. Save editable `.vsdx` in your internal working folder and keep this repo as source-of-truth for text-based diagrams.

## File naming

- `KUBRIC_7_Tier_Architecture.vsdx`
- `KUBRIC_SOC_Architecture.vsdx`
- `KUBRIC_NOC_Architecture.vsdx`
- `KUBRIC_Data_Flow.vsdx`
